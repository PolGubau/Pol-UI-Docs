The **StickyScroll** is a motion component that changes the color of the background when the user scrolls.

## Importation

Import the component from **pol-ui** to use the StickyScroll element:

```jsx
import { StickyScroll } from "pol-ui";
```

## Props

| Prop name | Type                        | Default | Description                                 |
| --------- | --------------------------- | ------- | ------------------------------------------- |
| colors    | string[]                    | -       | The colors to be used in the StickyScroll   |
| theme     | [StickyScrollTheme](#theme) | -       | The theme to be used in the StickyScroll    |
| className | string                      | -       | The class to be used in the StickyScroll    |
| children  | ReactNode                   | -       | The children to be used in the StickyScroll |
