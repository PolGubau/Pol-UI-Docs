The **Dropzone** displays a box where the user can drag and drop files to upload them. It also allows the user to click on the box to open the _file explorer_ and select files to upload.

## Importation

Import the component from **pol-ui** to use the Timeline element:

```jsx
import { Timeline } from "pol-ui";
```

## Props

### Timeline

| Prop name  | Type                    | Default | Description                               |
| ---------- | ----------------------- | ------- | ----------------------------------------- |
| theme      | [TimelineTheme](#theme) | {}      | The theme to be applied to the component. |
| horizontal | boolean                 | false   | The direction of the timeline.            |

### TimelineItem

| Prop name     | Type                        | Default | Description                               |
| ------------- | --------------------------- | ------- | ----------------------------------------- |
| theme         | [TimelineItemTheme](#theme) | {}      | The theme to be applied to the component. |
| iconContent   | ReactNode                   | null    | The content of the icon.                  |
| icon          | ReactNode                   | null    | The icon to be displayed.                 |
| iconClassName | string                      | null    | The class name of the icon.               |

## Theme

To learn more about how to customize the appearance of components, please see the [Theme docs](/docs/theming).

```ts
interface TimelineTheme {
  root: {
    direction: {
      horizontal: string;
      vertical: string;
    };
  };
  item: {
    root: {
      horizontal: string;
      vertical: string;
    };
    content: string;
    point: {
      horizontal: string;
      line: string;
      marker: {
        base: {
          horizontal: string;
          vertical: string;
        };
        icon: {
          base: string;
          wrapper: string;
        };
      };
      vertical: string;
    };
  };
}
```
