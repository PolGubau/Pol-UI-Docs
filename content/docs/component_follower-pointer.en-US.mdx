FollowerPointer is a component that allows you to create a pointer that follows the user's mouse.
The pointer is totally customizable and will affect the children components.

```jsx
import { FollowerPointer } from "pol-ui";
```

## Default FollowerPointer

<Example
  storybookUrl="followerpointer--default"
  height={300}
  name="FollowerPointer"
  code={root}
  embedUrl="followerpointer--default&viewMode=story"
>
  <RootCode />
</Example>

## Props

| Prop name | Type                           | Default   | Description                         |
| --------- | ------------------------------ | --------- | ----------------------------------- |
| className | string                         | undefined | Custom class name for the component |
| content   | string or React.ReactNode      | undefined | Content of the popup                |
| icon      | React.ReactNode                | undefined | Icon of the component               |
| theme     | [FollowerPointerTheme](#theme) | undefined | Theme of the component              |
| children  | React.ReactNode                | undefined | Children of the component           |

## Examples

### FollowerPointer with custom icon

To modify the icon of the **FollowerPointer** component, you can use the **icon** prop and pass any React component.

<Example
  storybookUrl="followerpointer--custom-icon"
  height={300}
  name="With custom icon"
  code={helperText}
  embedUrl="followerpointer--custom-icon&viewMode=story"
>
  <HelperTextCode />
</Example>

### Custom content

You can change as well the content prop to pass any string or React component.

<Example
  storybookUrl="followerpointer--custom-element"
  height={300}
  name="Custom content"
  code={allColors}
  embedUrl="followerpointer--custom-element&viewMode=story"
>
  <AllColorsCode />
</Example>
