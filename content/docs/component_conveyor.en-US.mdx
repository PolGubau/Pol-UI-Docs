The **Conveyor** component is a complete components that allows you to create a conveyor belt of elements. It can be used to create a carousel, a news ticker, or any other kind of conveyor belt.

The conveyor from **pol-ui** has a stable animation and a good performance as it uses CSS transitions.

## Importation

Import the component from **pol-ui** to use the conveyor element:

```jsx
import { Conveyor } from "pol-ui";
```

## Default conveyor

## Props

| Prop name    | Type                    | Default      | Description                                |
| ------------ | ----------------------- | ------------ | ------------------------------------------ |
| children     | React.ReactNode         | -            | The children of the conveyor.              |
| direction    | ConveyorDirection       | "horizontal" | The direction of the conveyor.             |
| speed        | ConveyorSpeed           | "normal"     | The speed of the conveyor.                 |
| pauseOnHover | boolean                 | false        | Pause the conveyor when the mouse is over. |
| theme        | [ConveyorTheme](#theme) | -            | The theme of the conveyor.                 |
| className    | string                  | -            | The class name of the conveyor.            |
