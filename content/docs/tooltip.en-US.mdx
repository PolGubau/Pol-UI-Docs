Use the **Tooltip** component from **Pol-ui** to indicate helpful information when hovering over an element such as a button or link to improve the UI/UX of your website.

## Importation

Import the component from **pol-ui** to use the Tooltip element:

```jsx
import { Tooltip } from "pol-ui";
```

## Props

### Tooltip

| Prop name | Type                   | Default            | Description                                      |
| --------- | ---------------------- | ------------------ | ------------------------------------------------ |
| active    | boolean                | false              | The state of the tooltip                         |
| children  | React.ReactNode        | -                  | The content of the tooltip                       |
| rounded   | RoundedSizes           | 'xl'               | The rounded size of the tooltip                  |
| theme     | [TooltipTheme](#theme) | -                  | The theme of the tooltip                         |
| color     | Colors                 | ColorsEnum.primary | The color of the tooltip                         |
| disabled  | boolean                | false              | The state of the tooltip                         |
| onClick   | (e) => void            | -                  | The function to call when the tooltip is clicked |

<Callout emoji="ðŸ’¡">
  You can also provide any prop comming from the [HTML **button**
  tag](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/button).
</Callout>
