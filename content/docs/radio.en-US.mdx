The **Radio** component is a component that allows the user to select one option from a set of options.

It has a crucial role in forms, as it allows the user to select one option from a set of options.
It can be used as a **uncontrolled** input or a **controlled** component, which means that it requires a value and an onChange prop to work properly.

## Importation

Import the component from **pol-ui** to use the Radio element:

```jsx
import { Radio } from "pol-ui";
```

## Props

| Prop name | Type                 | Default            | Description                                                          |
| --------- | -------------------- | ------------------ | -------------------------------------------------------------------- |
| theme     | [RadioTheme](#theme) | {}                 | The theme object to customize the appearance of the Radio component. |
| color     | Colors               | ColorsEnum.primary | The color of the Radio component.                                    |
| label     | string               | -                  | The label of the Radio component.                                    |

}
